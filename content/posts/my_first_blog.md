## 1、使用 Zola 搭建个人博客的体验

今天我终于用 Zola 成功搭建了自己的个人博客！这是一个基于 Rust 的静态网站生成器，速度非常快。

## 2、为什么选择 Zola

1. **极速构建** - 得益于 Rust 的高性能
2. **简单易用** - 单二进制文件，无需复杂环境
3. **功能丰富** - 支持代码高亮、数学公式等

## 3、安装过程

##  安装 Zola (我是自定义的templates| base.html|post.html|index.html自己构建的)

git clone https://github.com/getzola 

## 4、创建新项目
zola init 

## 5、构建新项目
zola build

## 6、 本地预览
zola serve
